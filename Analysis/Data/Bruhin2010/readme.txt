Included Files
--------------
- data.sqlite3:                Relational data base (SQLite3) containing all data used in the paper
- instructions.pdf:            Translation of the instructions used in the Zurich 2003 experiment
- supplementary.pdf:           Additional estimations and appendices
- XXXX_data.csv:               Data in comma separated form
- XXXX_startval_model1cl.csv:  Parameter estimates from the C=1 model
- finmix_estimation_routine.r: R routine for estimating the finite mixture models

Variables in the data sets
--------------------------
- exp:        Experiment indicator (1=CH03; 2=CN05; 3=CH06)
- id:         Individual identifier
- lottery:    Lottery identifier
- p1:         Probability of outcome z1
- z1:         Outcome z1 in local currency
- z2:         Outcome z2 in local currency
- ce:         Elicited certainty equivalent
- female:     Indicator (0=male; 1=female)
- semester:   Number of semesters enrolled
- highincome: Indicator for high income (explained in supplementary.pdf)
